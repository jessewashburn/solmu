# Utility Scripts

This directory contains one-off utility scripts used during development and maintenance.

## Scripts

- **check_all_field_lengths.py** - Checks field lengths in the database
- **check_instrumentation.py** - Validates instrumentation data
- **check_long_fields.py** - Identifies fields that exceed expected lengths
- **update_normalized_titles.py** - Updates normalized title fields in the database

These scripts are kept for reference and occasional maintenance tasks but are not part of the main application.
